# Darbų planas

## Tikslas
Paversti esamą app iš „AI art studio“ tipo MVP į aiškų, lengvai testuojamą **photo-to-coloring-page** produktą.

## 1 savaitė — migracija ir stabilizacija
- Perkelti kodą į GitHub
- Atidaryti projektą VS Code
- Paleisti lokaliai
- Sutvarkyti `.env`
- Patikrinti build / deploy
- Identifikuoti pagrindinius failus:
  - home screen
  - upload flow
  - generate flow
  - export / print flow
  - coloring flow

## 2 savaitė — UX refaktoringo bazė
- Pakeisti home screen copy
- `Blank Canvas` pakeisti į `Try Demo`
- Aiškūs CTA:
  - Upload Photo
  - Try Demo
- Paslėpti advanced controls
- Įvesti `Basic Mode`

## 3 savaitė — rezultatų patirtis
- Auto-fit generated image
- Aiškesni mygtukai:
  - Print Coloring Page
  - Download PDF
  - Download PNG
  - Color in App
- Before / After preview

## 4 savaitė — matavimas ir feedback
- Event tracking
- Error logging
- Feedback forma / modal
- Testas su 10–20 žmonių

## 5 savaitė — iteracija
- Pataisyti dažniausius friction taškus
- Supaprastinti labels
- Pagerinti empty states
- Pagerinti klaidų žinutes

## 6 savaitė — marketingo paruošimas
- Demo pavyzdžiai
- Sample gallery
- Landing page tekstas
- Beta test kvietimas
